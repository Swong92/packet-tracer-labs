# Packet Tracer Networking Labs

## Overview

This repository documents my hands on networking labs built using Cisco Packet Tracer. As a cybersecurity student, I created these labs to strengthen my understanding of networking fundamentals, routing, switching, and network security concepts.

The goal of this project is to simulate real-world networking environments and practice configuring and troubleshooting networks step-by-step.

Each lab contains:

* A network topology diagram
* Packet Tracer project files
* Configuration notes
* Documentation of what was learned

---

## Technologies Used

* Cisco Packet Tracer
* TCP/IP Networking
* Routing & Switching Concepts
* VLAN Segmentation
* DHCP & DNS Services
* Access Control Lists (ACLs)
* Network Troubleshooting
