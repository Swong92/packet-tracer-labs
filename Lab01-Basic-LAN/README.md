# Lab 01: Basic LAN Connectivity

## Objective

Create a simple LAN with two PCs connected to a switch and test connectivity.

## Network Topology

2 PCs connected to a switch.

## IP Addressing

PC1
192.168.1.10 /24

PC2
192.168.1.11 /24

## Steps

1. Add two PCs to the workspace
2. Add a 2960 switch
3. Connect both PCs to the switch using copper straight-through cables
4. Assign IP addresses
5. Test connectivity using ping

## Testing

Command used:

ping 192.168.1.11

Result: Successful connectivity between both devices.

## Skills Learned

* Basic LAN design
* IP configuration
* Network connectivity testing
* Switch-based communication
